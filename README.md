ECDSA signature vulnerability analysis library and CLI tool for research and key recovery.

## Features

- **Vulnerable Signature Generator**: Create test vectors with controlled weaknesses.
- **Multiple Attack Types**:
    - **Nonce Reuse**: Detects and exploits repeated nonces.
    - **Related Nonce**: Detects linear relationships between nonces ($k_{n+1} = k_n + \delta$ or $\alpha k_n$).
    - **Half-half Attack**: Single-signature recovery for nonces constructed from hash/key MSBs.
    - **LCG Detection**: Identifies nonces generated by weak PRNGs like LCG.
    - **Timing Detection**: Flags signatures with potential side-channel leaks.
- **Flexible Input/Output**: Supports JSON and CSV with human-readable or machine-parsable JSON output.

## Installation

```bash
cargo install --path .
```

## Usage

### 1. Generating Vulnerable Signatures

Use the `generate` subcommand to create test data for research:

```bash
# Generate 100 signatures with nonce reuse
vusi generate --weakness reuse --count 100 --output test.json

# Generate signatures with biased nonces (64-bit entropy)
vusi generate --weakness biased --bias-bits 64 --count 200

# Generate nonces using glibc LCG parameters
vusi generate --weakness lcg --seed 12345 --count 50
```

Supported weaknesses: `reuse`, `biased`, `lcg`, `related`, `polynonce`, `half-half`.

### 2. Analyzing Signatures

Analyze signatures to detect vulnerabilities and recover keys:

```bash
vusi analyze signatures.json
```

### 3. Piping Input

Generate and analyze in a single pipeline:

```bash
vusi generate --weakness half-half --count 1 | vusi analyze
```

## Attack Types

| Type | Signatures Required | Description |
|------|---------------------|-------------|
| **Nonce Reuse** | 2+ | Same $k$ value used for different messages. |
| **Related Nonce** | 2+ | Nonces follow a linear relationship ($k_{n+1} = k_n + \delta$). |
| **Half-half** | 1 | Nonce is $h_{msb} || d_{msb}$ (concatenated hash and key bits). |
| **LCG** | 2+ | Nonces generated by $k_{n+1} = (a \cdot k_n + c) \pmod n$. |
| **Timing** | 10+ | Statistical leaks in variable-time implementations. |

## Library Usage

```rust
use vusi::attack::{Attack, HalfHalfAttack};
use vusi::provider::load_signatures;

let signatures = load_signatures("signatures.json")?;
let attack = HalfHalfAttack;
let vulnerabilities = attack.detect(&signatures);

for vuln in vulnerabilities {
    if let Some(key) = attack.recover(&vuln) {
        println!("Recovered private key: {}", key.private_key_decimal);
    }
}
```
